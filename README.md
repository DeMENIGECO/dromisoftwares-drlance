# 🚀 DrLance — Estensione per Dromi

## ✨ Funzionalità

* 🎨 **Syntax Highlighting** (codice colorato)
* ⚡ **Riconoscimento automatico** dei file `.dr`
* 🧠 Base per **auto-completamento**
* ⚠️ Struttura pronta per **evidenziazione errori**
* 🔧 Configurazione editor ottimizzata per Dromi

---

## 💡 Perché usare DrLance?

<div align="left">
  <h3>🔥 L'estensione raccomandata</h3>
</div>

<div align="right">
  <h3>per sviluppare in Dromi su VS Code</h3>
</div>

Con **DrLance** puoi scrivere codice **Dromi** in modo più veloce, chiaro e organizzato dentro VS Code.

---

## 🌐 Sei nuovo in Dromi?

Trovi tutta la documentazione qui:

👉 https://demenigeco.github.io/dromilang/

## 📦 Installazione

1. Apri VS Code
2. Vai nella sezione Estensioni
3. Cerca **DrLance**
4. Installa e inizia a usare file `.dr`
